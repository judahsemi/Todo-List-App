# Todo-List-App

## Installation

**Clone repository**

`$ git clone https://github.com/judahsemi/todo-list-app.git`

**Change directory**

`$ cd todo-list-app`

**Install libraries**

`$ pip install -r requirements.txt`

## Run

**Start the server**

`$ python app.py`

**Vist link in your browser**

`http://127.0.0.1:5005/`
